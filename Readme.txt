Программа, которая из имеющегося массива строк формирует новый массив из строк, длина которых меньше, 
либо равна 3 символам. 

Алгоритм работы программы:
1. Ввод размера массива 
2. Ввод элементов массива
3. Создание массива
4. Подсчет кол-ва строк, удовлетворяющих условию (кол-во символов <= 3)
5. Создание номого массива с кол-вом элементов равному кол-ву символов, удовлетворяющих условию
6. Проверка исходного массива на выполнение условия, при успешном выполнении условия, присваиваем каждому элементу нового массива
элемент исходного
7. Вывод номого массива

Функция SizeArray просит пользователя указать кол-во элементов в массиве. 
Процедура CreateArray заполняет созданный массив строками, которые вводит пользователь.
Функция CountOfChar считает кол-во элементов, удовлетворяющих условию (кол-во символов <= 3).
Процедура ReleaseArray заполняет новый массив элементами исходонго массива, прошедшими проверку условия.
